# Description
The following files simulate the REST-based matchmaking feature of a 4-player multiplayer game built using FastAPI and Python Requests.

# Requirements
Terminal
Python
Python Libraries: fastapi, uvicorn, requests, pydantic

# Instructions

# RESTful API w/ FastAPI
Files - client_rest.py, server_rest.py

Open terminal.
Start server:                                       python3 server_rest.py
Open four more terminals.
For each terminal, run player clients:              python3 client_rest.py P1, python3 client_rest.py P2, ...,python3 client_rest.py P4
