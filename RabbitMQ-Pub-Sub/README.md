To get started with this project first go to setup.md and follow all the instructions so that
RabbitMQ works

after that in the terminal navigate this directory

and create an env (only need to create this one time)
    python3 -m venv .venv

to enter the env(need to enter every time when you open the project):
    source .venv/bin/activate

download pika(only need to do this the first time you setup the project)
    pip install pika
